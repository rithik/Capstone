# Capstone

The first time you set up the project, you need to run the following:

```bash
cd backend-node
yarn
cd ../frontend
npm install
```

To run server, run the following commands:

```bash
cd backend-node
node index
```

To run frontend, run the following commands:

- NPM: 
    ```bash
    cd frontend
    npm start
    ```

- Yarn:
    ```bash
    cd frontend
    yarn 
    ```
